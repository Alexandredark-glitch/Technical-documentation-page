Python Documentation

Introduction

Python is a high-level, interpreted programming language known for its simplicity, readability, and versatility. It supports multiple programming paradigms, including procedural, object-oriented, and functional programming. Python is widely used in web development, data science, automation, artificial intelligence, and more.

What You Should Already Know

Before diving into Python, it helps to be familiar with:

  • Basic programming concepts (variables, loops, functions)
  • Understanding of logic and control flow
  • Elementary mathematics
  • Experience with any programming language is a plus, but not required

Python and Other Languages

Python is often compared with JavaScript and Java:

  • Python vs JavaScript: Python emphasizes readability and simplicity, while JavaScript is mainly used for web development.
  • Python vs Java: Python syntax is simpler, and it doesn’t require explicit type declarations or compilation, unlike Java. Python is interpreted and dynamically typed.

Hello World

The classic first program:

print("Hello, World!")

Output:

Hello, World!

Variables

Variables are containers for storing data values. In Python, you don’t need to declare a type; it is inferred dynamically.

Declaring variables

x = 5 # Integer name = "Alice" # String pi = 3.14 # Float

Variable scope

  • Local variables: Defined inside a function and accessible only there.
  • Global variables: Defined outside any function and accessible throughout the program.
global_var = "I am global" def example(): local_var = "I am local" print(local_var) print(global_var) example()

Global variables

Use the global keyword to modify a global variable inside a function:

count = 0 def increment(): global count count += 1 increment() print(count) # Output: 1

Constants

Python doesn’t have true constants. By convention, variables with all uppercase names are treated as constants:

PI = 3.14159 GRAVITY = 9.81

If...else Statement

Conditional execution:

x = 10 if x > 0: print("Positive") elif x == 0: print("Zero") else: print("Negative")

While Statement

Repeated execution while a condition is true:

count = 0 while count < 5: print(count) count += 1

Output:

0 1 2 3 4
